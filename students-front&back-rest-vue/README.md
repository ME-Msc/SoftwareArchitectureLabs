# SoftwareArchitecture_hw3

Student system with vue &amp; rest

## run

- vue
  - cd students-vue
  - npm install
  - npm run serve
- rest
  - cd students-rest
  - mvn install
  - mvn spring-boot:run
- You will see the application on http://localhost:8080
- You could get the back-end data on http://localhost:8083 with "Postman"
